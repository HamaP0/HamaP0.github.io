## 💡 배운 점
1. git add [파일명]: 변경된 파일을 다음 커밋에 포함시키기 위해 준비(Staging)함.
2. git commit -m "[메시지]": 준비된 파일들을 저장소에 기록(커밋)함.
3. git push: 로컬 저장소의 변경 사항을 원격 저장소에 업로드함.
